# 🚀 MineMind

> **AI-Powered Smart Mining Management Platform**

MineMind is an AI-powered intelligent mining management platform that helps monitor, analyze, and optimize mining operations using Digital Twin technology, Causal Intelligence, Predictive Analytics, and Decision Support Systems.

🌐 **Live Demo:**  
https://minemind-457401289414.asia-southeast1.run.app

- 🏭 Smart Mining Dashboard
- 🌍 Live Digital Twin

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/ayushmeshram15/MineMind.git
```

Move into the project

```bash
cd MineMind
```

Install dependencies

```bash
npm install
```

Run the frontend

```bash
npm run dev
```

Run the backend

```bash
uvicorn main:app --reload
```
